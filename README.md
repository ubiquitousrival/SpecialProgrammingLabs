# Lab 3: OpenCV Filters

## Requirements
- OS: Ubuntu 20.04+
- Dependencies: cmake, g++, libopencv-dev

## Installation & Build
1. Run the build script:
   ./build.sh

## Usage
1. Run the app:
   ./run.sh

## Controls
- **Slider**: Adjusts brightness, blur intensity, or pixel size.
- **Keys**:
  - '1': Normal Mode
  - '2': Invert Colors
  - '3': Gaussian Blur (Interactive)
  - '4': Canny Edge
  - '5': Sobel Filter
  - '6': Mirror
  - '7': Sepia Effect (New)
  - '8': Pixelize Effect (New)
- **ESC / q**: Exit
